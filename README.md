# AI-Powered Smart Traffic Monitoring System

## Overview
A real-time AI-based traffic monitoring and violation detection system using Computer Vision, Deep Learning, and LLMs.

The system detects vehicles, tracks them using DeepSORT, identifies line-crossing violations, and generates AI-powered incident reports using Llama 3 via Ollama.

---

## Features
- Real-time vehicle detection using YOLOv8
- Multi-object tracking with DeepSORT
- Vehicle counting and analytics
- Line-crossing violation detection
- AI-generated incident reports using Llama 3
- GPU-accelerated inference using CUDA

---
